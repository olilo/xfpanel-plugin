Regular expression to apply on job name in order to show changed name on the dashboard.
In field 'Job name replace regular expression' put regular expression.
In field 'Job name replacement' put replacement text.
Example:

1.
Job name replace regular expression: prefix
Job name replacement: left empty
for job name prefix_my_job will give my_job

2.
Job name replace regular expression: prefix_(.*)
Job name replacement: $1_suffix
for job name prefix_my_job will give my_job_suffix


Basically it uses java String.replaceAll(regExp, replacement) on job name, so refer there for more explanation.
